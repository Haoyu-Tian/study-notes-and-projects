# Guess-Your-Number
C++ 实现的盲猜数字小游戏，基于数学原理“百发百中”。

## 项目简介
这是一个控制台版的数字读心术游戏：玩家记住一个 0~100 的数字，根据提示完成一系列加减乘运算，程序最后会直接揭晓你算出的最终数字。

核心原理：任何数乘 9 后，各位数字之和最终都会收敛到 9。

## 技术特点
- 纯 C++ 编写，Windows 控制台直接运行
- 彩色文字 + 倒计时，体验更流畅
- 超强输入容错，不会因乱输卡死
- 模块化结构，代码清晰易读

## 运行说明
1. 编译并运行 `GuessYourNumber.cpp`
2. 按提示完成计算，每次输完按 1 继续
3. 程序会输出你最终得到的数字
